# samples/app

This folder contains small generated wrapper fixtures for the repository.

The files here are produced by `tools/nextfs-entry.mjs` from `samples/nextfs.entries.json` and are kept intentionally small:

- `page.js`: sample `'use client'` page entry wrapper
- `actions.js`: sample `'use server'` action wrapper

Use this folder when you need a minimal, checked-in output fixture for the wrapper generator. For a runnable app layout, use [examples/nextfs-starter](../../examples/nextfs-starter/README.md).
